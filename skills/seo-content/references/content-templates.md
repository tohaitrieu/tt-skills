# Content Templates

## SEO Article Structure

```markdown
---
title: "{keyword} - {benefit/angle}"
slug: "{keyword-slug}"
description: "{150-160 chars với keyword}"
keywords: ["{primary}", "{secondary1}", "{secondary2}"]
category: "{category}"
author: "To Hai Trieu"
status: draft
created_at: "{YYYY-MM-DD}"
---

# {H1 với keyword}

{Hook paragraph - grab attention, state problem}

{Preview paragraph - what reader will learn}

## {H2 - Main Topic 1}

{Content with examples from VN market}

### {H3 - Subtopic}

{Detailed explanation}

## {H2 - Main Topic 2}

{Content}

## {H2 - Practical Application}

{Real examples, case studies}

## Kết luận

{Summary + CTA}

---
**Đọc thêm:**
- [Internal link 1]
- [Internal link 2]
```

## Word Count Guidelines

| Type | Words | Sections |
|------|-------|----------|
| Guide | 2000-2500 | 5-7 H2 |
| Tutorial | 1500-2000 | 4-6 H2 |
| Analysis | 1000-1500 | 3-5 H2 |
| News | 500-800 | 2-3 H2 |

## Writing Style

- Vietnamese với dấu đầy đủ
- Professional nhưng dễ hiểu
- Short paragraphs (2-3 sentences)
- Use bullet points, tables
- Bold key concepts
- Include practical examples
